Scrolling slider

.mwScroll to the sliders parent element
And everything just slides, including back to the start

Can be applied by simply adding the additional class: .mwScroll to the sliders .mwSlider element

Fade Slider

Fade Slider

Can be applied by simply adding the additional class:
.mwFade to the sliders parent element
This is the default slider action yes,
but specifying it will help keep track of what slider does what action especially if multiple sliders exist doing different transitions
Because of course, if it's worth doing, it's worth doing properly

Can be applied by simply adding the additional class: .mwFade to the sliders .mwSlider element

Auto Transition Slider

Auto Transition Slider

To make the slider auto transition simply add the class:
.mwAuto to the sliders parent element
This can be applied to the Fade Slider too

To make the slider auto transition simply add the class: .mwAuto to the sliders .mwSlider element

Pause on Hover Slider

Pause on Hover Slider

To allow the auto transition to be paused when hovering over the slider, simply add the class:
.mwHoverPause to the sliders parent element
Only use either this or the Auto Transition class, not both together (unknown concenques may follow otherwise)
This can be applied to the Scroll Slider too

To allow the auto transition to be paused when hovering over the slider, simply add the class: .mwHoverPause to the sliders .mwSlider element.

Only use either .mwHoverPause or .mwAuto class, not both together (unknown concenques may follow otherwise)
